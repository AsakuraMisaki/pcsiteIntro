  • CommonJS http://www.commonjs.org/

    ECMAScript规范没有为JavaScript定义一个标准库，而标准库对构建更广泛的应用程序很有用， CommonJS API将通过定义处理许多常见应用程序需求的API来填补这一空白，最终提供与Python，Ruby和Java一样丰富的标准库。目的是应用程序开发人员能够使用CommonJS API编写应用程序，然后跨不同的JavaScript解释器和主机环境运行该应用程序。

  • npm https://www.npmjs.com/

    npm是JavaScript的包管理器和世界上最大的软件注册表。

  • Node.js http://nodejs.cn/

    Node.js是基于ChromeV8-JavaScript引擎构建的JavaScript运行时，旨在构建可伸缩的网络应用程序。

  • Vue.js https://cn.vuejs.org/v2/guide/

    Vue是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。

  • webpack https://www.webpackjs.com/concepts/

    webpack 是一个现代 JavaScript 应用程序的静态模块打包器(module bundler)。当 webpack 处理应用程序时，它会递归地构建一个依赖关系图(dependency graph)，其中包含应用程序需要的每个模块，然后将所有这些模块打包成一个或多个 bundle(包)
